Responsive Web Design

April 23, 2017 | Assignment 2

Technology today is changing rapidly; it seems like every day we have a new way to view things online. Each time it seems like the screen grows, shrinks, or changes shape entirely. This poses a problem for any web designer or developer who is hoping for their users to view their content in a specific manner.

Unfortunately, because of the variety of devices these days, it is close to impossible for every user to have the same experience with a website or application, but we can get close. This is where the concept of Responsive Web Design (RWD) comes into play. The idea of RWD is that by using design and coding techniques that are flexible to changes in the viewport, we can optimize the viewers experience of our designs across most platforms and browsers.

Of course it is difficult to guarantee any user will have the exact same experience as others, but as web designers and developers utilizing the principles of responsive web design, we can ensure that we are doing everything we can to try and standardize the experience to the best of our ability. The wonderful thing about designing for user experience across platforms is that the techniques change practically every day, and this ensures that we have constantly have new methods for optimization.
